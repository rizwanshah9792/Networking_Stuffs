# Networking Stuffs

A collection of computer network assignments and projects covering various networking tools and concepts.

## Contents

This repository contains assignments related to:

- **Sockets** - Socket programming examples and exercises
- **iperf3** - Network performance measurement and bandwidth testing
- **hping3** - Packet crafting and network security testing
- **Wireshark** - Network packet analysis and protocol inspection

## Directory Structure

```
├── sockets/          # Socket programming assignments
├── iperf3/           # iperf3 network testing assignments
├── hping3/           # hping3 packet crafting assignments
└── wireshark/        # Wireshark packet analysis assignments
```

## Getting Started

Navigate to the relevant directory to find assignments for each topic. Each directory contains its own README with specific instructions and documentation.

## Requirements

- Python 3.x (for socket programming)
- iperf3
- hping3
- Wireshark
