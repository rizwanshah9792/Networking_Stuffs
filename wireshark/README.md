# Wireshark Assignments

This directory contains Wireshark packet analysis assignments.

## Topics Covered

- Packet capture and analysis
- Protocol inspection
- Network troubleshooting
- Traffic pattern analysis

## Prerequisites

- Wireshark installed on your system
- Network interface access for packet capture

## Installation

```bash
# Ubuntu/Debian
sudo apt-get install wireshark

# macOS
brew install --cask wireshark
```

## Usage

Each assignment includes its own documentation and usage instructions. Capture files (.pcap) may be included for analysis exercises.
