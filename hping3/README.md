# hping3 Assignments

This directory contains hping3 packet crafting and network security testing assignments.

## Topics Covered

- Packet crafting
- Port scanning techniques
- Firewall testing
- Network security analysis

## Prerequisites

- hping3 installed on your system
- Root/sudo privileges for raw socket access

## Installation

```bash
# Ubuntu/Debian
sudo apt-get install hping3
```

## Usage

Each assignment includes its own documentation and usage instructions.

**Note:** Use these tools responsibly and only on networks you have permission to test.
