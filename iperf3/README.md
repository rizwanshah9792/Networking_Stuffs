# iperf3 Assignments

This directory contains iperf3 network performance testing assignments.

## Topics Covered

- Bandwidth measurement
- TCP/UDP throughput testing
- Network performance analysis
- Jitter and latency measurement

## Prerequisites

- iperf3 installed on your system
- Network access for testing

## Installation

```bash
# Ubuntu/Debian
sudo apt-get install iperf3

# macOS
brew install iperf3
```

## Usage

Each assignment includes its own documentation and usage instructions.
